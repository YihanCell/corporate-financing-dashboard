# v0.3.2 - 仓库整理与命名简化

本版本把项目名称和仓库结构改得更直白、更适合业务同事理解。

## 主要更新

- 项目中文名统一为“企业融资情况数据看板”。
- GitHub 仓库名统一为 `corporate-financing-dashboard`。
- 页面标题、托盘名称、启动弹窗、README 和发布包命名全部同步。
- 根目录只保留 `start.bat` 和 `stop.bat` 两个常用入口。
- Windows 辅助脚本整理到 `scripts/windows/`。
- 示例 Excel 改名为 `samples/企业融资情况表示例.xlsx`。
- 删除旧的中文重复启动入口，降低选择成本。

# v0.3.1 - 项目命名与公开发布准备

本版本主要用于把项目整理成更适合公开 GitHub 仓库展示和 Release 分发的形态。

## 主要更新

- 项目中文名调整为“司库融资看板”。
- GitHub 仓库建议名调整为 `treasury-finance-monitor`。
- 页面标题和顶部品牌文案调整为 `Treasury Finance Monitor`。
- README 新增 About 段落，说明项目用途、运行方式和数据边界。
- Release 打包文件名调整为 `treasury-finance-monitor-版本号.zip`。
- 托盘图标、启动弹窗和防火墙规则名称统一为 `Treasury Finance Monitor`。

## 推荐使用

1. 下载并解压 `treasury-finance-monitor-v0.3.1.zip`。
2. 双击 `start_tray.bat`。
3. 右键右下角 `Treasury Finance Monitor` 托盘图标，打开局域网地址。
4. 在服务端页面中上传最新融资情况表。

# v0.3.0 - 托盘化与示例发布包

本版本把项目整理成更适合 GitHub Release 分发的形态。

## 主要更新

- 新增 Windows 托盘控制器：
  - 查看服务状态
  - 启动、停止、重启服务
  - 打开本机/局域网看盘
  - 复制局域网地址
  - 开启/关闭开机自启
- 新增稳定启动入口 `start_tray.bat`。
- 修复到期压力中 `8-30天`、`61-90天` 点击后筛选不稳定的问题。
- 新增示例融资情况表：
  - `samples/集团公司融资情况表示例.xlsx`
- 新增 README 截图：
  - `docs/images/dashboard-sample.png`
- 新增 Release 打包脚本：
  - `scripts/package_release.ps1`
- 新增 Python 依赖说明：
  - `requirements.txt`

## 推荐使用

1. 下载并解压 `treasury-finance-monitor-v0.3.0.zip`。
2. 双击 `start_tray.bat`。
3. 右键右下角 `Treasury Finance Monitor` 托盘图标，打开局域网地址。
4. 在页面中上传最新融资情况表。

## 示例数据

可以先导入 `samples/集团公司融资情况表示例.xlsx` 体验完整看盘效果。
